# Make vim copy and paste w xclip:

```
" Check if xclip is installed before setting up clipboard mappings
if executable('xclip')
	vnoremap <C-c> :w !xclip -selection clipboard<CR><CR>
	nnoremap <C-v> :r !xclip -o -selection clipboard<CR><CR>
endif

```

The <C-c> mapping is a visual mode mapping, which means you need to first enter visual mode (by pressing v in normal mode), select some text, and then press Ctrl-c to copy.

You're not in normal mode when trying to paste: The <C-v> mapping is a normal mode mapping, which means you need to be in normal mode (not insert mode or command-line mode) when you press Ctrl-v to paste.

" Function for toggling the bottom statusbar:
let s:hidden_all = 0
function! ToggleHiddenAll()
    if s:hidden_all  == 0
        let s:hidden_all = 1
        set noshowmode
        set noruler
        set laststatus=0
        set noshowcmd
    else
        let s:hidden_all = 0
        set showmode
        set ruler
        set laststatus=2
        set showcmd
    endif
endfunction
